# iStudy

Plataforma de estudos moderna e intuitiva.

## Tecnologias

- React 19.2.0
- TypeScript 5.9.3
- Vite 7.1.7
- Tailwind CSS 4.1.16

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
